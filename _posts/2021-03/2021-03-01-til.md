## 오늘 느낀 점
+ JPA를 처음 공부할 당시 JoinColumn이 어떻게 적용되는 건지 잘 이해가 안됬지만 대충 이해만 했다고 치고 넘어갔다. 그래서 그런지 역시나 복습할 때도 막혔다. 강의를 다시 찾아서 듣고 구글링하고 책을 뒤졌지만 확실하게 이해하지 못했다. 그래서 그냥 직접 왜 이럴까 하고 코딩해서 쿼리문이 나가는걸 확인해 봤다. 그랬더니 어떻게 동작하는지 이해할 수 있었다. 사실 코딩하는 과정에서 오류가 계속 떠서 왜 이런가 계속 고민하고 구글링하고 찾으나 너무 많은 시간을 사용했다. 이래서 혼자 공부하는게 참 힘들다..
+ SW 마에스트로 2차 코딩테스트를 봤다. 알고리즘 3개, SQL 1개, WEB 1문제가 나왔다. 알고리즘 2문제, SQL 1문제를 풀었다. 2문제를 남기고 1시간이 남았는데 알고리즘은 접근 방법이 떠오르지 못해서 포기했고 웹은 공부를 안해서 풀 수가 없었다. 그런데 웹 문제는 생각보다 엄청 간단해보였다. 조금만 공부하면 풀 수 있을정도? 느낌이었다. 사실 올해는 그냥 시험삼아 본 거라 아쉽지는 않다. 내년의 나는 확실히 다 풀 수 있을 것 같은 느낌이 든다.
