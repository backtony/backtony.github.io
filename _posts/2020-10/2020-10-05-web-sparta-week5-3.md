### 1. 실행
현재소스 실행   : Ctrl + Shift + F10  
최근소스 재실행 : Shift + F10  

### 2. 편집
되돌리기  : Ctrl + z  
다시 실행 : Ctrl + y  

### 3. 정렬
윈도우, 리눅스 : Ctrl + Alt + L  
Mac           : Command + Option + L

### 4. 디버깅
디버그        : Shift + F9  
한줄씩 실행   : F8  
한단계씩 실행 : F7
